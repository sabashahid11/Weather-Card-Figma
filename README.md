# 🌤️ Weather Card UI Design  

A clean, responsive weather card designed in Figma, featuring:  
- **City name**, **date**, and **hourly temperature forecast**  
- Mobile and desktop versions  
- Minimalist layout with optional weather icons

## Tools Used  
🛠️ Figma  

## How to Use  
1. Download the `.fig` file to edit in Figma.  
2. Export as PNG/JPG for development implementation.
